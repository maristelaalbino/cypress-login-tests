# 🧪 Testes Automatizados - Login

Projeto de automação de testes end-to-end utilizando Cypress.

## 📌 Objetivo
Validar o comportamento do fluxo de login de uma aplicação web, garantindo que diferentes cenários sejam tratados corretamente.

## ✅ Cenários testados
- Login com sucesso
- Senha inválida
- Usuário não cadastrado

## 🛠️ Tecnologias utilizadas
- Cypress

## ▶️ Como executar o projeto

1. Clone o repositório:
git clone https://github.com/SEU-USUARIO/NOME-DO-REPO

2. Instale as dependências:
npm install

3. Execute o Cypress:
npx cypress open

## 📚 Aprendizados
Durante o desenvolvimento deste projeto, foram aplicados conceitos de:
- Automação de testes E2E
- Interação com elementos (inputs, botões)
- Validação de elementos e mensagens na interface
